### ¿Cómo usarlo?.
#### Instalación

* Instalar Node.js v8.10.0 LTS
* Clonar este proyecto en tu equipo con `git clone git@github.com:OSWeekends/kienba.git`
* Ejecutar `npm install`
* Abre tu navegador en `https://localhost:3000`
